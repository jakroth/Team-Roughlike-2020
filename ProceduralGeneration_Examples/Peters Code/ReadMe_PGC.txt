Controls:

Dungeon:
G = Generate New Random Map
O = Generate New Random Set of Objects

W/A/S/D = move one tile over on the map (on press)
Up/Down/Left/Right = move one tile on the map (while button held downL)

Platformer:
G = Generate New Random Map
P = Generate alternate random platform sequence (requires G action first!!)

Left/Right = Apply force left/right
Space = Jump
